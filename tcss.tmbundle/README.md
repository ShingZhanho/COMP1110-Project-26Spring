# `tcss.tmbundle` - TextMate bundle for Textual CSS

This is a TextMate bundle for Textual CSS, the CSS dialect used in Textual.
This bundle can be added in JetBrains IDEs to provide proper syntax highlighting for `.tcss` files.

The language grammar is obtained from [Textualize/tcss-textmate-grammar](https://github.com/Textualize/tcss-textmate-grammar) and is licensed under the MIT License.
The `.yaml` file is converted to `.tmLanguage` using a [VSCode extension](https://marketplace.visualstudio.com/items?itemName=pedro-w.tmlanguage).
The `info.plist` was generated by an LLM.

## Installation

### For JetBrains IDEs

1. Go to PyCharm > Settings > Editor > TextMate Bundles.
2. Click the `+` button and select `tcss.tmbundle` from the file dialog.

### For Visual Studio Code

Use the official Textual CSS extension instead.
